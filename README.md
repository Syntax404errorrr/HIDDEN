## ğŸ§¾ About

HIDDEN is a Python-based project designed to simplify IPTV playlist management. It allows you to generate playlists, merge multiple `.m3u` files, and scrape playlist data efficiently. Perfect for enthusiasts who want clean, organized IPTV playlists.

---

## âœ… Features

- Generate `.m3u` playlists effortlessly.  
- Merge multiple playlist files into a single organized list.  
- Scrape or update playlist data using built-in scripts.  
- Supports custom playlist data via `playlist-data/`.  
- Handles both local and remote playlist sources.  
- Alphabetical sorting of channels for better navigation.
